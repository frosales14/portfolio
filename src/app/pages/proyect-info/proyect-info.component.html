Angular Country Explorer with REST Countries API (Fernando Herrera Course Project)

This Angular application, built as part of the Fernando Herrera Course "Angular de cero a experto" (From Zero to Expert), serves as a comprehensive country explorer powered by the REST Countries API. I've focused on enhancing the UI for a visually appealing and intuitive user experience. The app is fully responsive, adapting elegantly to any screen size for seamless exploration across devices.
